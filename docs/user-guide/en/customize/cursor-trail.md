# Cursor Trail

Display a trail effect when moving the cursor. Inspired by Ghostty terminal's cursor blaze feature.

---

## Enable

1. Open settings
2. Turn ON "Enable cursor trail" in the "Editor" section

---

## Features

- Trail fades out in 0.5 seconds after cursor movement
- Follows the theme's accent color
- Works with both Vim block cursor and bar cursor
- Automatically disabled on mobile devices
- Respects "Reduce motion" system preference
