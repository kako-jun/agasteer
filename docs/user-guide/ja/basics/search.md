# 検索

ホーム画面の検索ボックスで、ノート名・リーフ名・本文を横断検索できます。

---

## 検索の仕組み

1. 検索ボックスにキーワードを入力
2. 結果がリアルタイムで表示される
3. 検索結果をクリックすると、そのノートまたはリーフを開く
4. リーフ本文のヒットを開いた場合は、該当行へジャンプする

### 2ペイン表示時の振り分け

横長の2ペイン表示中は、検索結果の各行が左右に分割される。

- **左側をクリック** → 左ペインで開く（従来どおり）
- **右側（色の濃い領域）をクリック** → 右ペインで開く
- 左ペインに検索結果一覧を残したまま、右ペインで中身を確認する操作がしやすくなる

単一ペイン表示（スマホ縦など）では分割表示されず、行全体のクリックで現在のペインに開く。

---

## 検索の優先順位

1. **ノート名に一致** → 最優先で表示
2. **リーフ名に一致** → 次に表示
3. **本文に一致** → 最後に表示

---

## 検索対象

- 通常のノート・リーフ（Homeワールド）
- **Offlineリーフ**（ローカル専用メモ）も検索対象に含まれます
- **アーカイブ内のノート・リーフ**（アーカイブがロード済みの場合のみ）

> **ヒント**: アーカイブ内を検索するには、事前にGitHubからアーカイブをロードしておく必要があります。アーカイブからの検索結果は「アーカイブ/」というプレフィックスが付いて表示されます。アーカイブが未読み込みの場合は、検索結果の末尾にその旨が表示されます。
