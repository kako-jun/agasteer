# 検索

ホーム画面の検索ボックスで、ノート名・リーフ名・本文を横断検索できます。

---

## 検索の仕組み

1. 検索ボックスにキーワードを入力
2. 結果がリアルタイムで表示される

---

## 検索の優先順位

1. **ノート名に一致** → 最優先で表示
2. **リーフ名に一致** → 次に表示
3. **本文に一致** → 最後に表示

---

## 検索対象

- 通常のノート・リーフ（Homeワールド）
- **Offlineリーフ**（ローカル専用メモ）も検索対象に含まれます
- **アーカイブ内のノート・リーフ**（アーカイブがロード済みの場合のみ）

> **ヒント**: アーカイブ内を検索するには、事前にGitHubからアーカイブをロードしておく必要があります。アーカイブからの検索結果は「アーカイブ/」というプレフィックスが付いて表示されます。
