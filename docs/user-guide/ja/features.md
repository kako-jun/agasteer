# 便利な機能

> 知っておくと便利な機能を紹介します。

---

## 画面・表示

- **[2ペイン表示](./features/two-pane.md)** - 横長画面で左右分割、編集+プレビューの同時表示
- **[マークダウンプレビュー](./features/preview.md)** - Markdownを綺麗にレンダリング、画像でダウンロード

## 共有

- **[シェア機能](./features/share.md)** - URL・Markdown・QRコードで共有

## テキスト読み取り（OCR）

- **テキスト読み取り** - カメラ撮影またはファイル選択した画像からテキストを抽出し、リーフに挿入
  - リーフ編集中にフッター中央の📷ボタンから起動
  - ブラウザ内で完結（Tesseract.js）、外部サーバーへの送信なし
  - 日本語・英語に対応

## ファイル添付

- **[ファイル添付](./features/attachments.md)** - 画像・動画・音声・ZIPをリーフに添付
  - 貼り付け（スクショ直貼り）/ ドラッグ&ドロップ / フッターの添付ボタン
  - 画像は自動最適化（最大辺2048px・WebP変換、設定でOFF可）
  - 専用プライベートリポジトリ `{owner}/{repo}-media` に保存、オフラインでも添付できる

## データ管理

- **[アーカイブ](./features/archive.md)** - 使わなくなったノート/リーフを別の場所に保管
- **[インポート/エクスポート](./features/import-export.md)** - ZIPバックアップ、他アプリ（SimpleNote / Google Keep / Cosense (Scrapbox)）からの移行

## アプリ

- **[PWA対応](./features/pwa.md)** - スマホのホーム画面に追加してネイティブアプリのように使う
- **[初回ガイド](./features/tour.md)** - ウェルカムポップアップと吹き出しガイド

## キーボードショートカット

- **[ショートカット一覧](./features/shortcuts.md)** - 全ショートカットをまとめて確認

| ショートカット         | 動作                 |
| ---------------------- | -------------------- |
| **Ctrl+S** / **Cmd+S** | GitHubにPush（保存） |

> ブラウザの「ページを保存」ダイアログはブロックされます。

---

## トースト通知

各種操作の結果が画面下部に**トースト通知**として表示されます。

| 操作           | 通知例                     |
| -------------- | -------------------------- |
| Push成功       | 「保存しました」           |
| Pull成功       | 「データを取得しました」   |
| URLコピー      | 「クリップボードにコピー」 |
| インポート成功 | 「インポートしました」     |
| エラー発生     | エラー内容が表示される     |

通知は数秒で自動的に消えます。
