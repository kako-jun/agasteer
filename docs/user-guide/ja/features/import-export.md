# インポート/エクスポート

データのバックアップや、他アプリからの移行ができます。

---

## エクスポート

すべてのノート・リーフをZIPファイルでダウンロード。

### 手順

1. 設定画面を開く
2. 「**エクスポート**」ボタンをタップ
3. `agasteer-export-YYYY-MM-DD.zip` がダウンロードされる

### ZIPの中身

```
agasteer-export-YYYY-MM-DD/
├── notes/
│   ├── ノート1/
│   │   ├── リーフ1.md
│   │   └── サブノート/
│   │       └── リーフ2.md
│   └── ノート2/
│       └── リーフ3.md
└── metadata.json  # 並び順、バッジ情報など
```

### 注意

- Git履歴（.git/）は含まれない
- 初回Pull完了後にエクスポート可能

> **💡 Tip**: 定期的にエクスポートしておくと、万が一のときに安心。

---

## インポート

エクスポートファイルを取り込めます。

### 手順

1. 設定画面を開く
2. 「**インポート**」をタップ
3. ファイルを選択

### 対応フォーマット

| フォーマット            | ファイル形式                                     |
| ----------------------- | ------------------------------------------------ |
| Agasteer形式            | エクスポートしたファイル                         |
| SimpleNote形式          | .json（またはそれを含む.zip）                    |
| Google Keep形式         | Google Takeout の .zip、または解凍後の個別 .json |
| Cosense (Scrapbox) 形式 | プロジェクトのエクスポート .json                 |

### インポート後の挙動

- ソースごとのノート（`SimpleNote_1` / `GoogleKeep_1` / `Cosense_1`）が作成され、その中にリーフがフラットに配置される
- 先頭に「インポート結果」レポートリーフが生成される（何を取り込み、何をスキップしたかの記録）
- 非対応の要素はスキップされ、レポートに記録される

### Google Keepの注意点

- チェックリストは `- [ ]` / `- [x]` に変換
- リンク（annotations）は本文末に追記
- 画像添付・色分け・ラベル・ピン留めは非対応（レポートに記録）
- HTMLファイルと画像ファイルは使用しない（JSONのみ）
- `isTrashed: true` のノートはスキップ

### Cosense (Scrapbox) の注意点

Cosense にはZIP形式のエクスポートがないため、プロジェクトのエクスポート `.json` をそのまま選択してください。

**変換される記法**:

- `[URL 説明]` / `[説明 URL]` → `[説明](URL)` のMarkdownリンク
- `[画像URL]`（.png / .jpg / .jpeg / .gif / .webp / .svg） → `![](URL)` の画像
- `[URL]`（非画像単独） → `<URL>` の自動リンク
- `[ページ名]`（URLを含まない） → 解決先がないため `[ページ名]` のまま保持
- `#tag` → プレーンテキストとしてそのまま保持
- 行頭のインデント（空白・タブ） → アウトライン構造を失わないよう保持

**取り込まれないもの（レポートに記録）**:

- 外部画像のホスト側（URLは保持されるが表示可否は保証されない）
- ページのサムネイル画像（`page.image`）
- ページのビュー数
- 行ごとの編集者・タイムスタンプ
- ハッシュタグ `#tag` の内部リンク解決
- `[ページ名]` 形式の内部ページリンクの解決

### 同名のものがある場合

既存のノート/リーフと名前が重複する場合、**インポート開始前に確認ダイアログ**が表示されます。

**選択肢**:

| 選択           | 動作                                                                                 |
| -------------- | ------------------------------------------------------------------------------------ |
| **キャンセル** | インポートを中止                                                                     |
| **スキップ**   | 既存を優先し、インポートしない                                                       |
| **追加**       | ノート：既存ノートの中にリーフを追加<br>リーフ：自動リネームして追加（例: `メモ_2`） |

> **💡 Tip**: 同じファイルを2回インポートしても、「スキップ」を選べば重複しません。

### 注意

- 初回Pull完了後にインポート可能
- インポート後はPushして保存することを推奨
