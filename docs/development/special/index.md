# 特殊リーフ

通常のリーフとは異なる動作をする仮想リーフ。

## ドキュメント一覧

| ファイル                     | 内容                                                 | 行数 |
| ---------------------------- | ---------------------------------------------------- | ---- |
| [priority.md](./priority.md) | Priorityリーフ（[1]〜[5]マーカー集約、読み取り専用） | ~290 |
| [offline.md](./offline.md)   | Offlineリーフ（ローカル専用、GitHub同期なし）        | ~100 |

## 比較

| 項目           | Priority               | Offline                   |
| -------------- | ---------------------- | ------------------------- |
| **リーフID**   | `__priority__`         | `__offline__`             |
| **表示位置**   | ホーム先頭（order: 0） | ホーム最上位（order: -1） |
| **保存先**     | なし（仮想）           | IndexedDB                 |
| **編集**       | 読み取り専用           | 編集可能                  |
| **コンテンツ** | 自動生成               | ユーザー入力              |
| **統計**       | 除外                   | 除外                      |
| **バッジ**     | 設定可（非永続）       | 設定可（永続）            |
