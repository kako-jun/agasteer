# ADR（Architecture Decision Record）

Agasteerにおける主要な**不可逆設計判断**の記録です。

## このディレクトリについて

ここに置かれているADRは、いずれも実装が既に完了した後にIssue #228の対応として**遡及的に記録したもの**です。決定当時にADRを書く運用がなかったため、当時のコミットメッセージ・CHANGELOG.mdの記述を一次情報として裏取りした上で作成しています。

各ADRには、決定の根拠となった実際のコミット・CHANGELOG記述への参照を含めています。

今後、同種の不可逆な設計判断（バックエンド選定、状態管理方式、データ構造の大規模変更等）を行う際は、実装と同じタイミングでこのディレクトリに新しいADRを追加してください。
